# AI 文件导出指南

## 📝 关于 AI 文件

`.ai` 文件是 Adobe Illustrator 的源文件格式，**不能直接在网页中使用**。你需要将它导出为网页可用的格式。

## ✅ 推荐导出格式

### 1. SVG 格式（最推荐）⭐

**优点：**
- 矢量图，无损缩放
- 文件小，加载快
- 支持透明背景
- 可以在代码中修改颜色

**导出步骤：**
1. 在 Adobe Illustrator 中打开你的 `.ai` 文件
2. 选择菜单：`文件` → `导出` → `导出为...`
3. 选择格式：**SVG** 或 **SVG (svg)**
4. 点击 `导出`
5. 在导出选项中：
   - **样式**：选择 "内联样式" 或 "演示文稿属性"
   - **字体**：选择 "转换为轮廓"（确保字体正确显示）
   - **图像**：选择 "嵌入"（如果 logo 包含图片）
   - **对象ID**：选择 "图层名称" 或 "最小"
6. 点击 `确定`
7. 将导出的 `.svg` 文件保存为 `logo.svg`

### 2. PNG 格式（备选）

**优点：**
- 兼容性好
- 支持透明背景

**缺点：**
- 位图，放大可能模糊
- 文件通常比 SVG 大

**导出步骤：**
1. 在 Adobe Illustrator 中打开你的 `.ai` 文件
2. 选择菜单：`文件` → `导出` → `导出为...`
3. 选择格式：**PNG**
4. 点击 `导出`
5. 在导出选项中：
   - **分辨率**：选择 **高 (300 ppi)** 或 **屏幕 (72 ppi)**
     - 如果只是网页使用，72 ppi 足够
     - 如果需要高质量，选择 300 ppi
   - **背景色**：选择 **透明**
   - **抗锯齿**：勾选（让边缘更平滑）
6. 点击 `确定`
7. 将导出的 `.png` 文件保存为 `logo.png`

## 📐 尺寸建议

### SVG 格式
- 不需要指定尺寸，SVG 是矢量图会自动缩放
- 建议在 Illustrator 中设置合适的画板大小（如 512x512 像素）

### PNG 格式
- **导航栏使用**：建议 64x64 像素或 128x128 像素（2x 分辨率）
- **通用尺寸**：512x512 像素（可以满足各种使用场景）
- **高分辨率**：1024x1024 像素（如果需要更高质量）

## 🎨 颜色建议

由于你的网站使用**深色背景**，建议：

1. **浅色 Logo**（推荐）
   - 白色或浅色 Logo
   - 在深色背景上更清晰可见

2. **深色 Logo**
   - 如果 Logo 是深色的，可能需要调整
   - 或者使用 CSS 滤镜来反转颜色

## 📁 文件命名

导出后，将文件命名为：
- `logo.svg`（如果导出为 SVG）
- `logo.png`（如果导出为 PNG）

然后放到项目的 `public` 文件夹中。

## 🔄 完整流程

1. **在 Illustrator 中导出**
   - 打开 `.ai` 文件
   - 导出为 SVG 或 PNG
   - 保存到本地

2. **复制到项目**
   ```bash
   # 如果是 SVG
   cp /path/to/logo.svg /Users/xiaoqingran/Desktop/EsLatin/eslatin-official-website/public/logo.svg
   
   # 如果是 PNG
   cp /path/to/logo.png /Users/xiaoqingran/Desktop/EsLatin/eslatin-official-website/public/logo.png
   ```

3. **如果使用 PNG，更新代码**
   - 如果导出的是 PNG 而不是 SVG，需要告诉我，我会更新代码中的文件路径

4. **提交并部署**
   ```bash
   git add public/logo.svg  # 或 logo.png
   git commit -m "Add custom logo"
   git push origin main
   ```

## 💡 额外提示

### SVG 优化建议
- 导出后可以使用在线工具优化 SVG 文件大小
- 推荐工具：https://jakearchibald.github.io/svgomg/

### 多尺寸版本
如果需要不同尺寸的 Logo（如 favicon），可以导出多个版本：
- `logo.svg` - 主 Logo
- `logo-32x32.png` - 小图标（32x32）
- `logo-64x64.png` - 中等图标（64x64）

### 测试
导出后，在浏览器中打开 SVG/PNG 文件，检查：
- 是否显示正确
- 背景是否透明
- 颜色是否符合预期

## ❓ 常见问题

**Q: 可以直接使用 .ai 文件吗？**
A: 不可以。浏览器无法直接显示 .ai 文件，必须导出为 SVG 或 PNG。

**Q: SVG 和 PNG 哪个更好？**
A: SVG 更好，因为它是矢量图，可以无损缩放，文件也更小。

**Q: 如果 Logo 是彩色的可以吗？**
A: 可以，但建议使用浅色或白色 Logo，在深色背景上更清晰。

**Q: 导出后需要修改代码吗？**
A: 如果导出为 SVG，代码已经配置好了，不需要修改。如果导出为 PNG，需要更新代码中的文件路径。
