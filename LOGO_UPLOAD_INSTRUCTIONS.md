# Logo 上传说明

## 📁 如何上传你的 Logo

### 步骤 1: 准备 Logo 文件

请准备以下格式的 Logo 文件：
- **推荐格式**: SVG（矢量图，最佳质量）
- **备选格式**: PNG（透明背景，建议 512x512 像素或更高）

### 步骤 2: 上传文件

将你的 Logo 文件放到以下位置：

```
public/logo.svg
```

或者如果是 PNG 格式：

```
public/logo.png
```

### 步骤 3: 文件命名

- **SVG 格式**: 命名为 `logo.svg`
- **PNG 格式**: 命名为 `logo.png`

### 步骤 4: 文件要求

- **尺寸**: 建议至少 512x512 像素（SVG 格式会自动缩放）
- **背景**: 透明背景（PNG 格式需要）
- **颜色**: 建议使用浅色或白色 Logo（因为网站是深色背景）

## 📝 已修改的代码位置

代码已经更新为使用图片 Logo，替换了原来的图标。Logo 会显示在：

1. **导航栏** (`components/navbar.tsx`)
   - 位置：左上角，品牌名称旁边
   - 尺寸：32x32 像素

2. **页脚** (`app/page.tsx`)
   - 位置：页脚左侧，品牌名称旁边
   - 尺寸：24x24 像素

## 🔄 如果使用 PNG 格式

如果你上传的是 PNG 文件而不是 SVG，需要修改以下文件：

### 修改 `components/navbar.tsx`
将第 28 行的：
```tsx
src="/logo.svg"
```
改为：
```tsx
src="/logo.png"
```

### 修改 `app/page.tsx`
将第 223 行的：
```tsx
src="/logo.svg"
```
改为：
```tsx
src="/logo.png"
```

## ✅ 验证

上传文件后：
1. 刷新浏览器页面
2. 检查导航栏左上角是否显示你的 Logo
3. 检查页脚是否显示你的 Logo

## 💡 提示

- 如果 Logo 显示不正确，检查文件路径是否正确
- 确保文件在 `public` 文件夹中
- SVG 格式推荐，因为可以无损缩放
- 如果 Logo 颜色太深，可能需要调整颜色以适应深色背景
